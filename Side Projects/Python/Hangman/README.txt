This is a hangman game with the following features

	• Infinitely allows you to play
	• Allows you to guess word by either guessing letters or word
	• Keeps track of used letters
	• Shows word to guess in following format: _ _ _ _ _
	• Keeps track of the number of correct and wrong answers per play instance

Future developments

	• Keep track of all correct and wrong answers of all plays
	• Interactive image of man
